# 🎮 Cloud Console - Software Engineering Project

> A university Software Engineering case study based on a Cloud Gaming Platform.

---

## 📖 Overview

Cloud Console is a conceptual cloud gaming platform developed as a Software Engineering academic project. The project demonstrates the complete Software Development Life Cycle (SDLC) by modeling a real-world cloud gaming service through software engineering artifacts such as Software Requirement Specification (SRS), UML diagrams, ER diagrams, sequence diagrams, use case diagrams, activity diagrams, and other design documents.

The goal of this project is to apply Software Engineering principles to design a scalable, secure, and user-friendly cloud gaming ecosystem.

---

## 🎯 Project Objectives

* Understand Software Engineering concepts through a real-world case study.
* Prepare complete Software Requirement Specifications (SRS).
* Design UML models for the proposed system.
* Develop database models using ER diagrams.
* Model system behavior using Sequence, Activity, and Use Case diagrams.
* Demonstrate SDLC documentation for an academic project.

---

## 💡 Project Description

Cloud Console is a cloud-based gaming platform that enables users to access and play games remotely without requiring high-end hardware. The platform manages cloud gaming sessions, subscriptions, user accounts, game libraries, achievements, cloud saves, and administrative operations.

This repository focuses on the **analysis and design phase** of Software Engineering rather than the implementation of a real cloud gaming infrastructure.
